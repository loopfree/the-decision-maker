# The Decision Maker
> This is a WebApp which will hopefully help you to make a decision by answering your question with yes, no, or maybe

# Built With
* HTML
* SASS
* Vue.js

# Getting Started

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
